## ❌ Problem Statement

**200 Million Indians have money to invest — but don't.**

| Stat | Insight |
|------|---------|
| **200M+** | Indians with disposable income who don't invest |
| **5,000+** | Mutual funds create decision paralysis |
| **3%** | Indian equity participation vs 55% in the US |

### Root Causes
- 📚 **Financial Jargon Barrier** — NAV, CAGR, XIRR, SIP, ELSS are alien vocabulary for 90% of Indians
- 😵 **Decision Paralysis** — 5,000+ MFs, 5,000+ stocks, FDs, gold, PPF, NPS — zero guidance on what fits YOU
- 🤖 **Tools, Not Companions** — Groww, Zerodha, Paytm Money let you buy/sell but never *guide* you
- 🏘️ **Tier 2/3 City Neglect** — 80% of fintech focus is on metro users; small-town India needs Hinglish guidance

## 💡 Our Solution

**Nivesh Saathi** is a **Jarvis-level AI companion** — not just a tool — that:

1. **Onboards via natural chat in Hinglish** — no forms, understands your life in a 2-minute conversation
2. **Auto-creates a personalized Money Plan** with 3 smart investment buckets
3. **Tracks EVERYTHING** — investments, expenses, assets, liabilities, insurance, life events, tax
4. **Studies the market** — real-time Nifty, Sensex, Gold, sector analysis with sentiment scoring
5. **Generates AI signals** — buy/sell/hold/rebalance with confidence scores and full reasoning
6. **Pushes proactive notifications** — sector opportunities, tax deadlines, spending warnings
7. **Provides daily AI briefs** — morning market + portfolio snapshot + action items
8. **Calculates net worth** — Assets − Liabilities = real-time financial health
9. **Gamifies investing** — Money Garden where goals grow from seed 🌰 to fruit tree 🌳✨

## ✨ Key Features — 17 AI-Powered Features

### 💬 Conversational Onboarding
Chat-based setup in Hinglish. No boring forms — Saathi asks naturally about income, expenses, goals, and risk appetite. Generates a full investment plan in under 2 minutes.

### 🪣 3-Bucket Money System
| Bucket | Purpose | Risk | Expected Return |
|--------|---------|------|-----------------|
| 🟢 **Safe Pocket** | Emergency Fund, Liquid Funds, FD | Low | ~7% p.a. |
| 🟡 **Growth Pocket** | SIP, Mutual Funds, Goal-Based | Medium | ~14% p.a. |
| 🔴 **Opportunity Pocket** | High Growth, Stocks, Crypto | High | ~20% p.a. |

Auto-allocated based on user's risk profile (Safe / Balanced / Aggressive).

### 💰 Investment Tracker
Record, edit, and delete investments across all three buckets. Link investments to specific goals. Full history with bucket-wise and goal-wise breakdowns.

### 📊 Expense Tracker
Categorized spending tracking (Food, Transport, Shopping, Bills, Entertainment, Healthcare, Education, Other) with AI-generated saving insights and spending pattern analysis.

### 🧠 AI Insights Engine
Portfolio health score, risk mismatch warnings, diversification analysis, emergency fund status, rebalancing alerts — all powered by real portfolio data, zero assumptions.

### 📈 Market Intelligence
Real-time data on Nifty 50, Sensex, Gold prices, FD rates, RBI repo rate, CPI inflation. Sector-wise trend analysis (IT, Banking, Pharma, Auto, etc.) with market sentiment scoring (Fear/Neutral/Greed).

### 🤖 Saathi Companion (Jarvis Engine)
The crown jewel — a full financial AI companion with 6 tabs:
- **Feed** — AI-generated proactive notifications
- **Daily Brief** — Personalized morning briefing
- **Signals** — Buy/Sell/Hold/Rebalance signals with confidence %
- **Net Worth** — Assets − Liabilities dashboard
- **Assets** — Track MFs, stocks, gold, FD, PPF, NPS, real estate, crypto
- **Watchlist** — Track instruments with target buy/sell prices

### 🔔 AI Notification Engine
Proactive, context-aware alerts generated from your actual portfolio data:
- *"Tech stocks down 3% — good entry point for your growth bucket"*
- *"Tax-saving deadline in 45 days — invest ₹40K in ELSS"*
- *"Monthly spend up 22% — dining out grew ₹4,500"*

### 🎯 Goal-Based Planning
8 pre-built goal templates (Home, Education, Car, Travel, Emergency, Wealth, Retirement, Wedding) + custom goals. Auto-calculates monthly SIP needed with visual progress tracking.

### 🌱 Money Garden (Gamification)
Each goal is a plant that grows as you invest:
- 🌰 **Seed** (0%) → 🌱 **Sprout** (25%) → 🌿 **Sapling** (50%) → 🌳 **Tree** (75%) → 🌳✨ **Fruit Tree** (100%)
- Creates emotional connection — 3x better retention vs traditional dashboards.

### 💱 SIP Calculator
Interactive calculator with growth visualization. Slide inputs for amount, tenure, and returns to see projected wealth.

### 📚 Learn Hub
Bite-sized financial education modules in Hinglish covering SIP, mutual funds, tax saving, market basics, and more.

### 📉 Buy/Sell/Hold Signals
AI-generated signals with:
- Signal type (Buy 📈 / Sell 📉 / Hold ✋ / Watch 👁️ / Rebalance 🔄 / Alert ⚠️ / Opportunity 💡 / Tax Action 💸)
- Confidence score (0-100%)
- Full reasoning for every signal

### 📋 Personalized Daily Brief
Morning greeting + market snapshot + portfolio overnight performance + top 3 action items + motivational nudge.

### 💳 Net Worth Tracker
Complete financial dashboard:
- **Assets**: Mutual Funds, Stocks, Gold, FD, PPF, EPF, NPS, Real Estate, Crypto
- **Liabilities**: Home Loan, Car Loan, Personal Loan, Credit Cards, Education Loan
- **Net Worth** = Total Assets − Total Liabilities (with monthly trend)

### 👀 Watchlist
Track stocks, mutual funds, ETFs, indices, gold, and crypto with current prices, change %, and target buy/sell price alerts.

### 💸 Tax Optimizer
Section 80C, 80D tracking. Old vs New regime comparison. Tax-saving investment suggestions based on actual portfolio gaps.

## 🚀 How to Run

### Prerequisites
- Node.js (v18+)
- npm or yarn
- Expo CLI (`npm install -g expo-cli`)
- Expo Go app on your phone (for testing)

### Steps

```bash
# 1. Clone the repository
git clone <repo-url>
cd nivesh-saathi

# 2. Install dependencies
npm install

# 3. Start the Expo dev server
npx expo start

# 4. Run on your device
# Scan the QR code with Expo Go (Android) or Camera (iOS)

# 5. Or run on specific platform
npx expo start --android
npx expo start --ios
```

### Troubleshooting
- If Expo Go shows a white screen with *"Open App.js to start working on your app"*, ensure your `index.ts` entry point is correctly configured
- Make sure all devices are on the same WiFi network
- Run `npx expo start --clear` to clear cache if needed

```bash

npx expo run:android
```
